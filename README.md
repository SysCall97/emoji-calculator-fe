# Emoji Calculator Frontend

## Installation
 Step-1: 
```bash
 $ git clone https://github.com/SysCall97/emoji-calculator-fe.git
```
 Step-2:
 ```bash
 $ npm install
```
Step-3:<br/>
Setup and run laravel backend from 
```bash
https://github.com/SysCall97/emoji-calculator-be
```
 Step-4:
 ```bash
 $ npm start
```
